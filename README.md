## Install

Macos instructions:

```bash
git --version  # Will prompt you to install development tools
bash -c "$(curl -fsSL https://raw.githubusercontent.com/dermoumi/dotfiles/HEAD/setup.sh)" -- -l -c -a
```

Ubuntu instructions:

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/dermoumi/dotfiles/HEAD/setup.sh)" -- -l -c -a
```
